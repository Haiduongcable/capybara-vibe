## 2. Setup Free AI Subscription

If you want to use **Free AI subscriptions** (such as OpenAI Codex or Google Antigravity) instead of paying for API usage, you will need to download **ProxyPal**.

> **ProxyPal** is the native desktop application version of CLIProxyAPI. It acts as a local proxy server that manages authentication and connection to free AI providers for you.

### Step 1: Download ProxyPal

*   **macOS (Apple Silicon):** [ProxyPal_0.3.42_aarch64.dmg](https://github.com/heyhuynhgiabuu/proxypal/releases/download/v0.3.42/ProxyPal_0.3.42_aarch64.dmg)
*   **Ubuntu/Linux:** [ProxyPal_0.3.42_amd64.deb](https://github.com/heyhuynhgiabuu/proxypal/releases/download/v0.3.42/ProxyPal_0.3.42_amd64.deb)
*   **Windows:** [ProxyPal_0.3.42_x64-setup.exe](https://github.com/heyhuynhgiabuu/proxypal/releases/download/v0.3.42/ProxyPal_0.3.42_x64-setup.exe)

### Step 2: Install & Configure

After downloading, open the ProxyPal app and follow the video guides below to set up your subscription.
